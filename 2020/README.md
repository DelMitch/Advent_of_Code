# Advent of Code 2020
Language: **Python3**

## Project Structure
<pre>
2020/  
    |---- input_files/  
        |----> aoc_xx.txt  
    |---- utils/  
        |----> __init__.py  
        |----> input_handler.py  
        |----> output_handler.py  
    |---- day_xx/  
        |----> part_a.py  
        |----> part_b.py
</pre>

## How to Run day_xx part_x
Prerequisite: install python3  
1. Navigate to the 2020 directory via commandline  
2. `python3 -m day_xx.part_x`  
Example: for day\_01 part\_a, it would be `python3 -m day_01.part_a`
